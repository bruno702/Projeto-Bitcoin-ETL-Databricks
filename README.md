#  Monitoramento Automatizado

Este projeto realiza o monitoramento automatizado do preço do Bitcoin utilizando dados em tempo real da API pública da Coinbase.

O sistema consulta periodicamente o valor atual do ativo, identifica se houve alta ou baixa em relação à última cotação registrada e também verifica a cotação atual do dólar.

Com base nessas informações, o sistema gera alertas automáticos por e-mail e atualiza um dashboard analítico desenvolvido no Databricks para acompanhamento visual das variações do mercado.

---

##  Funcionalidades

- Consulta automática do preço atual do Bitcoin  
- Consulta da cotação atual do dólar  
- Identificação de variação (alta ou baixa)  
- Envio automático de alerta por e-mail  
- Atualização de dashboard analítico  

---

##  Objetivo do Projeto

O objetivo deste projeto é demonstrar habilidades em:

- Consumo de APIs REST  
- Automação de processos  
- Monitoramento de dados financeiros em tempo real  
- Integração com ambiente analítico (Databricks)  
- Organização de código para portfólio profissional
